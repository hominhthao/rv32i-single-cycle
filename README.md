# RV32I Single-Cycle CPU

A SystemVerilog implementation of a RV32I single-cycle processor.

## Features

- Full Adder
- 32-bit Ripple Carry Adder
- ALU
- Register File
- Immediate Generator
- Branch Comparator
- Control Unit
- Instruction Memory
- Load Store Unit
- RV32I Single-Cycle Datapath

## Tools

- SystemVerilog
- Icarus Verilog
- Surfer
- VSCode
- TerosHDL
